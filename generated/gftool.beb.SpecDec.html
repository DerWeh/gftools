gftool.beb.SpecDec¶

class gftool.beb.SpecDec(rv: np.ndarray, eig: np.ndarray, rv_inv: np.ndarray)[source]¶

SVD like spectral decomposition.

Works only for N×N matrices unlike the UDecomposition base class.
